## 📺 المشهد 1️⃣: التحضير الأولي

### ما تحتاجه:
- ✅ Node.js (من https://nodejs.org/)
- ✅ المتصفح (Chrome, Firefox, Safari... أي واحد)
- ✅ ملفات المشروع (من outputs)

### التحقق من التثبيت:

افتح Terminal/Command Prompt واكتب:

```bash
node --version
```

لو شُفت رقم مثل `v18.0.0` ✅ تمام!

## 📺 المشهد 7️⃣: إصلاح المشاكل (اللي قد تواجهك)

### ❌ المشروع ما يشتغل

**التشخيص:**
```bash
npm run dev
```

**الحل:**
```bash
npm install --legacy-peer-deps
npm run dev
```

### ❌ Terminal يقول "npm: command not found"

**الحل:** ثبّت Node.js من:
👉 https://nodejs.org/

ثم أغلق Terminal وافتحه جديد

### ❌ المتصفح يقول "Connection refused"

**التشخيص:**
- تأكد من وجود Terminal مفتوح
- تأكد من الأمر `npm run dev` يشتغل

**الحل:**
```bash
Ctrl+C  # أوقف الخادم
npm install
npm run dev  # شغّل مرة ثانية
```

### ❌ الأيقونات ما تظهر

**الحل:**
```bash
npm install lucide-react@latest
```

ثم اضغط F5 في المتصفح

## 📺 المشهد 8️⃣: التعديلات والتطوير

### لو تبي تعدّل حاجة:

1. **افتح الملف** في أي محرر نصوص (VSCode مثلاً)
2. **عدّل الكود**
3. **احفظ الملف** (Ctrl+S)
4. **شوف المتصفح** - يتحديث تلقائياً! ✨

### مثال:
لو تبي تغيير رقم الهاتف:
```bash

src/data/gifts.js

# وعدّل:
phone: "01038395688"  # الرقم الحالي
# إلى:
phone: "01234567890"  # رقمك الجديد

# احفظ واشوف التغيير فوراً!
```
