# ⚡ Quick Start - شغّل المشروع في 3 دقائق!

## الخطوة 1️⃣ (30 ثانية)
اضغط Windows+R واكتب `cmd` ثم اضغط Enter

أو على Mac: Cmd+Space → اكتب `Terminal`

---

## الخطوة 2️⃣ (1 دقيقة)

انسخ والصق هالأمر في Terminal:

```bash
npm --version
```

لو ظهر رقم (9.0.0 أو أعلى) ✅ كويس
لو ما ظهر ❌ حمّل Node.js من https://nodejs.org/

---

## الخطوة 3️⃣ (30 ثانية)

اذهب لمجلد المشروع:

```bash
cd Desktop/giftoly-store
```

(غيّر المسار حسب مكان المجلد عندك)

---

## الخطوة 4️⃣ (1-2 دقيقة)

اكتب:

```bash
npm install
```

انتظر لما تخلص... ☕

---

## الخطوة 5️⃣ (30 ثانية)

اكتب:

```bash
npm run dev
```

---

## الخطوة 6️⃣ (انتهينا! 🎉)

افتح المتصفح واذهب لـ:

```
http://localhost:5173/
```

---

## 🎊 خلص! المشروع شغّال!

الآن يمكنك:
- 🛍️ تصفح الهدايا
- 🔍 البحث عن منتجات
- 🛒 إضافة للسلة
- 💬 التواصل عبر WhatsApp

---

## 🆘 مشكلة؟

### المشروع ما يشتغل؟
```bash
Ctrl+C
npm install
npm run dev
```

### Terminal يقول "npm: command not found"؟
ثبّت Node.js: https://nodejs.org/

### المتصفح يقول خطأ؟
اضغط Ctrl+Shift+R في المتصفح

---

## 🛑 لو تبي توقف المشروع:
اضغط `Ctrl+C` في Terminal

## 🔄 لو تبي تشغله من جديد:
```bash
npm run dev
```

---

**That's it! استمتع بالمشروع!** 🚀✨
