# 🎁 Giftoly - متجر الهدايا الفاخرة

متجر الكتروني حديث متخصص في بيع الهدايا الفاخرة والمختارة بعناية.

## ✨ المميزات الرئيسية

- 🛍️ **كتالوج الهدايا الفاخرة** - تشكيلة متنوعة من الهدايا المميزة
- 🔍 **البحث والتصفية** - ابحث عن الهدية المثالية بسهولة
- 🛒 **سلة التسوق** - إضافة والتحكم في كمية المنتجات
- 💬 **تواصل مباشر** - دردشة WhatsApp فورية مع خدمة العملاء
- 📱 **تصميم متجاوب** - يعمل بسلاسة على جميع الأجهزة
- 🎨 **واجهة جميلة** - تصميم عصري بألوان جذابة

## 🛠️ المتطلبات

- Node.js 16+ 
- npm أو yarn

## 📦 التثبيت

```bash
# استنسخ المشروع
git clone https://github.com/yourusername/giftoly-store.git

# انتقل إلى المجلد
cd giftoly-store

# ثبّت المكتبات
npm install
```

## 🚀 تشغيل المشروع

```bash
# في وضع التطوير
npm run dev

# سيفتح الموقع تلقائياً على http://localhost:5173
```

## 🏗️ البناء للإنتاج

```bash
# بناء النسخة الإنتاجية
npm run build

# اختبار النسخة المبنية
npm run preview
```

## 📁 هيكل المشروع

```
giftoly-store/
├── src/
│   ├── components/        # المكونات القابلة لإعادة الاستخدام
│   ├── pages/            # صفحات التطبيق
│   ├── context/          # إدارة الحالة (Cart Context)
│   ├── data/             # البيانات الثابتة
│   ├── App.jsx           # المكون الرئيسي
│   ├── index.jsx         # نقطة البداية
│   └── index.css         # الأنماط العامة
├── public/               # الملفات الثابتة
├── index.html           # صفحة HTML الرئيسية
├── vite.config.js       # إعدادات Vite
├── tailwind.config.js   # إعدادات Tailwind CSS
└── package.json         # معلومات المشروع والمكتبات
```

## 🛠️ التقنيات المستخدمة

- **React 19** - مكتبة بناء الواجهات
- **Vite** - أداة البناء السريعة
- **Tailwind CSS** - إطار عمل التصميم
- **Lucide React** - مكتبة الأيقونات
- **Context API** - إدارة الحالة

## 📝 الصفحات الرئيسية

### 🏠 الصفحة الرئيسية (Home)
- عرض جميل للعلامة التجارية
- مميزات المتجر
- دعوة للعملاء لاستكشاف الهدايا

### 🎁 صفحة الهدايا (Gifts)
- عرض جميع الهدايا المتاحة
- البحث والتصفية حسب الفئات
- عرض تفاصيل المنتج والسعر

### 🛒 صفحة السلة (Cart)
- عرض المنتجات المضافة
- تعديل الكميات
- حساب السعر الإجمالي
- إرسال الطلب عبر WhatsApp

### 📞 صفحة التواصل (Contact)
- معلومات التواصل المختلفة
- ساعات العمل
- حسابات وسائل التواصل الاجتماعي
- الأسئلة الشائعة
