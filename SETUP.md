# 🚀 دليل سريع للتشغيل

## الخطوة 1️⃣: تثبيت المكتبات

```bash
npm install
```

هذا سيثبت جميع المكتبات المطلوبة من:
- React 19.2.6
- React DOM 19.2.6
- Lucide React 1.16.0
- Tailwind CSS 4.3.0
- Vite 5.0.8

## الخطوة 2️⃣: تشغيل المشروع

```bash
npm run dev
```

سيفتح الموقع تلقائياً على:
```
http://localhost:5173
```

## الخطوة 3️⃣: الاستمتاع! 🎉

المشروع جاهز الآن! يمكنك:
- ✨ استكشاف الهدايا الفاخرة
- 🔍 البحث عن المنتجات
- 🛒 إضافة للسلة
- 💬 التواصل عبر WhatsApp

---

## 📦 بناء النسخة الإنتاجية

عندما تريد نشر المشروع:

```bash
npm run build
```

سيُنشئ مجلد `dist` يحتوي على النسخة الجاهزة للنشر.

---

## ⚙️ المشاكل الشائعة والحلول

### المشكلة: الأيقونات ما تظهر
✅ **الحل:** تأكد من تثبيت `lucide-react`
```bash
npm install lucide-react@latest
```

### المشكلة: Tailwind CSS ما يشتغل
✅ **الحل:** فعّل خادم التطوير وانتظر بشوية
```bash
npm run dev
```

### المشكلة: الواتساب ما يفتح
✅ **الحل:** تأكد من أن رقم الهاتف صحيح في `src/data/gifts.js`

---

## 📚 ملفات مهمة

- `src/App.jsx` - المكون الرئيسي
- `src/index.css` - الأنماط
- `src/data/gifts.js` - بيانات الهدايا والتواصل
- `tailwind.config.js` - إعدادات Tailwind
- `vite.config.js` - إعدادات Vite
